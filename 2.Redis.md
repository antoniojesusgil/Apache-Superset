### Redis Cli
`redis-cli` is the Redis command line interface, a simple program that allows to send commands to Redis, and read the replies sent by the server, directly from the terminal.

```
> redis-cli
127.0.0.1:6379> ping
PONG
127.0.0.1:6379>
```
